# 🕒 Relógio Digital

Um projeto de relógio digital moderno e funcional, desenvolvido para praticar a integração entre estrutura, estilo e dinamismo na web.

## 🎯 Foco do Projeto
O objetivo principal deste projeto foi o fortalecimento da **base em JavaScript**. Foquei em entender como a linguagem orquestra a lógica por trás da interface, aplicando conceitos fundamentais como:

* **Manipulação do DOM:** Alteração de elementos HTML em tempo real.
* **Objeto `Date`:** Captura e tratamento de dados de tempo do sistema.
* **Funções de Intervalo:** Uso do `setInterval` para atualização contínua da interface.
* **Lógica de Formatação:** Tratamento de strings para exibição correta de horas, minutos e segundos.

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estrutura semântica.
* **CSS3:** Estilização e layout responsivo.
* **JavaScript:** Lógica principal e dinamismo.
